# Binary Search Tree Implementation in C++

## Overview
This project implements a simple Binary Search Tree (BST) in C++. The BST supports the following operations:
- Insert(int value)
- Search(int value)
- Delete(int value)
- Inorder Traversal
- Preorder Traversal
- Postorder Traversal
- Levelorder Traversal
- Count the number of nodes
- Find the height of the tree

## Files
- BinaryTree.h: Contains the class definition and method declarations.
- BinaryTree.cpp: Implements the methods for the Binary Search Tree.
- main.cpp: Contains the main function for testing the BST operations.
- README.txt: Instructions on how to compile and run the code.

## Compilation and Execution
To compile the code, use the following command:
```
g++ -o bst main.cpp BinaryTree.cpp
```

To run the compiled program, use:
```
./bst
```
